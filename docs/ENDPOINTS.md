# Available Endpoints

> ## **GET** /machines

Gets a list of all machines and details about them

> **HEADERS**  

- **Authorization** : Bearer {Token}

> **PARAMETERS**

***NONE***

> **RESPONSE**

> ## **Delete** /user

Deletes a user 

> **HEADERS**  

- **Authorization** from Single Sign On 

> **PARAMETERS**

- ***UserID***

> **RESPONSE**
- *Successfully deleted user*

> ## **GET** /user

Gets information about a user

> **HEADERS**  

- **Authorization** from Single Sign On

> **PARAMETERS**

- ***UserID***

> **RESPONSE**
- *Returns information about the User*
